# Прототип системы детекции подхода к столам в предприятии общественного питания (может использоваться для контроля уборки столов)

Упрощённый, но рабочий прототип, демонстрирующий основной пайплайн детекции событий на примере одного столика в видеозаписи.

---

## Установка зависимостей

```bash
pip install -r requirements.txt
```

> `ultralytics` автоматически скачает веса `yolov8n.pt` при первом запуске.

---

## Запуск

```bash
python main.py --video NAME.mp4
или python main.py --video "название видео.mp4"

После запуска откроется окно с **первым кадром** видео - нужно **выделить зону столика мышью**.
